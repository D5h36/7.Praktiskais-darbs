Praktiskais darbs Nr.7.
Tēma: Attēli. Karte.
Uzdevumi:
1. Versiju kontrole (3 pt).
⎯ Darbu veikt izmantojot Git versiju kontroli. Izmantot Git
mitināšanas pakalpojumu (piemēram, GitHub) nav obligāti.
⎯ Repozitorijs nedrīkst sekot (track) attēliem vai cita liela izmēra
bināriem objektiem (BLOBs). Nevēlamo datņu ignorēšanai pielietot .gitignore.
⎯ Saglabāt vismaz 3 versijas.

2. Attēlu galerija (3 pt).
⎯ Atrast vai izveidot vismaz 7 dažādu izmēru attēlus. Attēlu saturs nav
būtisks. Izpētīt un izvēlēties tīmeklim piemērotu failu tipu.
⎯ Ievietot attēlus responsīvos vienāda izmēra konteineros. Attēli
nedrīkst tikt stiepti, t.i. tiem jāsaglabā proporcija. Attēla malas drīkst nebūt
redzamas.

3. Karte (2 pt).
⎯ Izpētīt risinājumus kartes ievietošanai mājaslapā.
⎯ Pievienot karti ar marķieri kādā lokācijā.

4. Veikt HTML validāciju ar W3C validator https://validator.w3.org,
novērst visus brīdinājumus un kļūdas (2 pt).
